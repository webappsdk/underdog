Demo: Client side javascript plugin framework

This demo will show how client side plugin framework works, it illustrates the article: Plugin architecture II: Client side javascript plugin framework. The code of the demo can be downloaded here. This demo uses underdog.js library.

A first plugin is fired when document is loaded, it will print the list with the articles' titles. When user clicks a title the content of the article is printed below. When the content of an article is printed two other plugins are fired: One that will add a textbox for searching words in the text (it will highlight matched words), and another that will identify the word with more than 3 characters under the user's mouse and will show a tooltip.
